# SegNet-PyTorch

This project is a implementation of [SegNet](https://arxiv.org/abs/1511.00561) by PyTorch, You can find more detail by visit the demo [website](http://mi.eng.cam.ac.uk/projects/segnet/) provided by authors.


### Environment
* ubuntu: 16.04
* PyTorch: 0.4.0
* python: 3.6.1

### Run the project
sh run.sh
